题解 P3950 部落冲突

link-cut tree 板子题

这道题可以用来作为link-cut tree的练手题

C操作:把发生战争的俩部落的连边cut掉

U操作:把停战的俩部落link起来

Q操作:如果p部落和q部落在一棵树里(树根相同)，就输出”Yes”,否则输出”no”
